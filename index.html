about

Genuary is an artificially generated month of creative coding — every January, a new prompt each day challenges artists and developers to build things that are beautiful, weird, or unexpected with code.


This portfolio collects all 31 of my sketches for Genuary 2026. Each piece was built from scratch, in a single day, following that day's prompt.


→ genuary.art

✕
tech

All sketches are built with vanilla JavaScript and the HTML5 Canvas API — no external libraries, no frameworks.


Some days use WebGL / GLSL shaders for GPU-accelerated generative work. A few experiments lean on SVG for vector output.


The portfolio shell itself is plain HTML + CSS, designed to evoke a classic pixel-art OS desktop.
